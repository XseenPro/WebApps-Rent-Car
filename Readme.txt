------Export database ke phpmysql-----

Pertama install xampp https://www.apachefriends.org/download.html

1. BUKA XAMPP AKTIFKAN APACHE & MYSQL
2. BUKA http://localhost/phpmyadmin
3. buat database rental
4. import rental.sql
5. jalankan flask run

------How to run-----

1. install python in https://python.org/
2. install flask dengan perintah "pip install flask" dalam cmd
3. masuk ke virtualenv "venv\Scripts\activate" dalam cmd
4. install mysql dengan cara "pip install flask-mysql" dalam cmd setelah cara ke-3
5. ketik "set FLASK_APP=app.py"
6. ketik "flask run"
